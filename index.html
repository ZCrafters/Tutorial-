Apache ✅ Running

1 Buat Database "Perpustakaan"

Perintah SQL:

MariaDB [(none)]> CREATE DATABASE Perpustakaan;
Query OK, 1 row affected (0.001 sec)
💝 Penjelasan buat Elen:
- CREATE DATABASE = perintah buat database baru
- Perpustakaan = nama database (bebas, tapi ikuti soal ya)
- Kalau muncul "Query OK" berarti SUKSES! ✅
2 Aktifkan Database

Perintah SQL:

MariaDB [(none)]> USE Perpustakaan;
Database changed

MariaDB [Perpustakaan]> ← Nanti promptnya berubah jadi ini
💝 Penjelasan buat Elen:
- USE = pilih database yang mau dipakai
- Perhatikan prompt berubah dari [(none)] jadi [Perpustakaan]
- Ini tandanya Elen sudah masuk ke database Perpustakaan
3 Buat Tabel Anggota

Perintah SQL:

MariaDB [Perpustakaan]> CREATE TABLE Anggota (
    id_anggota INT PRIMARY KEY,
    nama VARCHAR(50),
    alamat VARCHAR(100),
    nomor_telpon VARCHAR(15)
);
Query OK, 0 rows affected (0.023 sec)
💝 Penjelasan buat Elen:
- CREATE TABLE = buat tabel baru
- INT = tipe data angka
- VARCHAR(50) = teks maksimal 50 huruf
- PRIMARY KEY = kunci utama (unik, tidak boleh sama)
- Jangan lupa tutup kurung dan titik koma di akhir!

Cek hasilnya:

MariaDB [Perpustakaan]> DESC Anggota;

| Field | Type | Null | Key | Default | Extra |

| alamat | varchar(100)| YES | | NULL | |
| nomor_telpon | varchar(15) | YES | | NULL | |
+----------------+-------------+------+-----+---------+-------+
4 rows in set (0.001 sec)
4 Buat Tabel Buku

Perintah SQL:

MariaDB [Perpustakaan]> CREATE TABLE Buku (
    kode_buku INT PRIMARY KEY,
    judul VARCHAR(100),
    pengarang VARCHAR(50),
    stok INT
);
Query OK, 0 rows affected (0.019 sec)
💝 Penjelasan buat Elen:
- Sama seperti tabel Anggota, tapi nama kolomnya berbeda
- stok INT = jumlah buku (angka)
5 Tambah Kolom Email di Tabel Anggota

Perintah SQL:

MariaDB [Perpustakaan]> ALTER TABLE Anggota ADD email VARCHAR(50);
Query OK, 0 rows affected (0.012 sec)
Records: 0 Duplicates: 0 Warnings: 0
💝 Penjelasan buat Elen:
- ALTER TABLE = mengubah tabel yang sudah ada
- ADD = menambah kolom baru
- Kolom email ditambahkan di tabel Anggota
