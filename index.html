Rumus Dasar SQL yang Wajib Elen Hafal:
CREATE → INSERT → SELECT → UPDATE → DELETE

📋 Petunjuk Pengerjaan untuk Elen:

  1. Buka XAMPP → Start Apache & MySQL
  2. Buka browser → ketik localhost/phpmyadmin
  3. Klik menu "SQL" untuk mengetik perintah
  4. Atau buka Command Prompt (CMD) → ketik: mysql -u root
  5. Setelah selesai, screenshot hasilnya!
1 Buat Database dengan nama Perpustakaan
CREATE DATABASE Perpustakaan;
Elen, perintah CREATE DATABASE digunakan untuk membuat database baru. Nama databasenya adalah Perpustakaan. Jangan lupa titik koma (;) di akhir ya!
2 Aktifkan Database Perpustakaan
USE Perpustakaan;
Elen, perintah USE digunakan untuk memilih database yang akan digunakan. Ini seperti Elen masuk ke ruangan Perpustakaan, supaya bisa kerja di sana.
3 Buat table Anggota (id_anggota, nama, alamat, nomor_telpon)
CREATE TABLE Anggota ( id_anggota INT PRIMARY KEY, nama VARCHAR(50), alamat VARCHAR(100), nomor_telpon VARCHAR(15) );
Elen, perintah CREATE TABLE untuk membuat tabel baru. INT = angka bulat, VARCHAR = teks. PRIMARY KEY = kunci utama (unik, tidak boleh sama).
4 Buat table Buku (kode_buku, judul, pengarang, stok)
CREATE TABLE Buku ( kode_buku INT PRIMARY KEY, judul VARCHAR(100), pengarang VARCHAR(50), stok INT );
Elen, sama seperti soal 3, tapi nama tabelnya Buku dan kolomnya berbeda. stok pakai INT karena angka (banyaknya buku).
5 Tambahkan kolom/field email pada table Anggota
ALTER TABLE Anggota ADD COLUMN email VARCHAR(50);
Elen, perintah ALTER TABLE untuk mengubah struktur tabel yang sudah ada. ADD COLUMN = menambah kolom baru. Di sini nambah kolom email.
6 Isi 3 data pada table Anggota & 3 data pada table Buku

Supratman', '08112783738', 'jupi@gmail.com'); -- Data 3: Elen (Kamu!) INSERT INTO Anggota VALUES (103, 'Elen', 'Jl. Poltangan', '0813048393', 'elen@gmail.com');

Data untuk Tabel Buku:

-- Data 1 INSERT INTO Buku VALUES (111, 'Basis Data Untuk Pemula', 'Ade Supriatna', 4); -- Data 2 INSERT INTO Buku VALUES (222, 'Ekonomi Syariah', 'Bintang Gemilang', 7); -- Data 3 INSERT INTO Buku VALUES (333, 'Desain Grafis', 'Sudiyono', 9);
Elen, perintah INSERT INTO untuk memasukkan data. VALUES diikuti oleh data sesuai urutan kolom. Teks pakai tanda petik ('), angka tidak perlu petik.

7 Ubah nomor telpon Elen menjadi 0811111111
UPDATE Anggota SET nomor_telpon = '0811111111' WHERE nama = 'Elen';
Elen, perintah UPDATE untuk mengubah data. SET = nilai yang baru, WHERE = kondisi (yang mana yang diubah). Hati-hati! Kalau lupa WHERE, semua data akan berubah!
8 Ubah nama pengarang Ekonomi Syariah menjadi Bintang Purnama
UPDATE Buku SET pengarang = 'Bintang Purnama' WHERE judul = 'Ekonomi Syariah';
Elen, sama seperti soal 7, tapi tabelnya Buku. Kita cari buku yang judulnya 'Ekonomi Syariah', lalu ganti pengarangnya.
9 Tampilkan judul dan pengarang dengan stok terbanyak
SELECT judul, pengarang FROM Buku WHERE stok = (SELECT MAX(stok) FROM Buku);
Elen, ini sedikit rumit. Kita pakai SELECT di dalam SELECT (subquery). MAX(stok) = cari nilai stok paling besar. Hasilnya: Desain Grafis karena stoknya 9 (paling banyak).

Atau cara lebih simpel (kalau hanya 1 yang terbanyak):

SELECT judul, pengarang, stok FROM Buku ORDER BY stok DESC LIMIT 1;
Cara ini: urutkan dari stok terbesar (DESC), lalu ambil 1 saja (LIMIT 1). Lebih mudah dipahami ya Elen!
10 Hapus buku dengan judul Desain Grafis
DELETE FROM Buku WHERE judul = 'Desain Grafis';
Elen, perintah DELETE FROM untuk menghapus data. Jangan lupa WHERE ya! Kalau lupa WHERE, semua data di tabel akan terhapus!
